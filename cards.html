Lógica de programação

A lógica de programação tem como objetivo mostrar métodos para solucionar problemas, além de determinar a sequência lógica para o desenvolvimento de um programa. É criada a partir de um conjunto de elementos como: organização, perseverança, padronização, criatividade e otimização.

Algoritmo

Um algoritmo consiste em uma série limitada de etapas, cujo objetivo é executar uma determinada tarefa. É uma série de instruções ordenadas de maneira lógica para solucionar determinado problema. Para a sua construção, temos três estruturas: sequenciação, seleção e repetição. Além disso, precisamos dividir o problema em questão em três etapas: entrada, procesamento e saída.

Programa

Um programa é um algoritmo criado segundo as regras de sintaxe de uma linguagem de programação. Consiste em uma sequência de comandos que apresenta as tarefas a serem realizadas para atingir a solução de um determinado problema.

Linguagem de programação

A linguagem de programação é um método padronizado, formado por um conjunto de regras sintáticas e semânticas, de implementação de um código fonte - que pode ser compilado e transformado em um programa de computador, ou usado como script interpretado - que informará instruções de processamento ao computador.
